# Lab Recap and Conclusion

## Congratulations

Congratulations on completing this learning lab!

You took a major step forward in your programmability journey by learning how to use Ansible with Cisco IOS XE APIs across multiple model-driven interfaces.

## What You Completed

In this lab, you built and validated an end-to-end workflow that included:

- Environment and tooling setup for Ansible API automation
- NETCONF-based configuration using Cisco native models
- RESTCONF-based automation using native and IETF-oriented workflows
- gNMI-based configuration with Cisco native models
- gNMI-based configuration with OpenConfig models for portability

## Core Skills You Strengthened

You now have hands-on experience with:

- Building reusable Ansible playbooks for API-driven operations
- Separating read-only confidence checks from configuration changes
- Working with YANG model paths and structured payloads
- Comparing protocol and model choices for real automation tasks
- Verifying outcomes through repeatable, automation-first methods

## Why This Matters

By learning these patterns, you are moving from one-off configuration tasks to consistent, scalable network automation practices.

This is exactly the skill progression needed to build reliable infrastructure workflows in modern Cisco IOS XE environments.

## Final Note

Great work! Keep building on this momentum and continue exploring what programmable infrastructure can do with Ansible and Cisco IOS XE.
